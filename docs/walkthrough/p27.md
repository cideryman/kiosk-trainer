## 🌟 구현 핵심 기능 요약

### 1. P27-1: 발달장애인 AAC 그림 답글 프리셋 복구 및 연동 (`reviews.html`)
- **AAC 프리셋 복구**: Git 커밋 누락으로 인해 유실되어 있었던 AAC 답글 프리셋 기능(즉시 등록, 락 제어, 테두리 강조 활성화)을 완벽히 재구현했습니다.
- **원터치 즉시 등록**: 프리셋 칩 클릭 시 별도의 답글 저장 버튼 클릭 없이 즉시 백엔드 DB로 답글이 전송 및 저장됩니다.
- **시각적 강조 피드백 (`.btn-reply-preset.active`)**: 현재 선택되었거나 이미 등록 완료된 답글 텍스트와 매칭되는 프리셋 버튼에 초록색 강조 테두리를 렌더링합니다.
- **중복 제출 방지 (Locking)**: 답글 전송이 처리되는 중에는 모든 프리셋 버튼이 50% 투명도로 비활성화되어 연타에 의한 중복 제출을 원천 차단합니다.

### 2. P27-2: TTS 이모티콘 필터링 및 청각 안내 최적화
- **이모티콘 필터링 헬퍼 함수 (`stripEmojis`)**: 정규식을 활용하여 텍스트 내 유니코드 이모티콘(💖, 👍, 🛵, 🌟 등)과 특수 기호를 깔끔하게 발라내 주는 정제 모듈을 내장했습니다.
- **노이즈 없는 청각 피드백**: 
  - 프리셋 버튼 클릭 시 이모티콘(예: 💖)을 읽어주는 불필요한 음성(예: "반짝반짝 하트")을 제외하고, 핵심 본문 문구("응원 고마워요!")만 명료하게 한국어로 읽어줍니다.
  - 후기 읽어주기(P26) 시에도 손님 후기 본문에 삽입된 기호나 스탬프 명칭에 포함된 유니코드 그림 문자를 배제하고 한글만 낭독하여 인지적 피로감을 크게 개선했습니다.
