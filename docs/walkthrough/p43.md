# P43 행사 모드 커스텀 엠블럼 설정(Base64) 워크스루

## 개요

주방(Kitchen) 화면에서 배달왔삼(Guest)의 기본 상단 엠블럼(오토바이 아이콘)을 행사용 커스텀 이미지로 변경할 수 있도록 구현했습니다. 구글 드라이브 폴더를 별도로 만들지 않고, **웹 브라우저 단에서 이미지를 최적화(Max 150x150, WebP)하여 Base64 텍스트 형태로 설정 시트(Settings)에 직접 저장**하는 초경량·고속 아키텍처를 적용했습니다.

---

## 주요 변경 내역

### 1. 주방 엠블럼 업로드 및 최적화 UI (`kitchen.html`, `js/kitchen.js`)
- `🎯 게스트 운영 메뉴 모드` 섹션에 **[사진 변경]** 및 **[초기화]** 버튼과 썸네일 미리보기를 제공하는 UI 추가.
- 모바일 가독성을 고려해 행사명 입력창 바로 아래에 깔끔하게 정렬.
- `handleEmblemUpload()` 함수 구현:
  - HTML5 `<canvas>`를 사용해 원본 이미지를 **가로세로 최대 150px 비율 유지** 상태로 리사이징.
  - 품질 0.8의 **WebP 포맷**으로 인코딩하여 Base64 문자열(`guestEventEmblemBase64`) 추출.
- `saveGuestSettings()` 호출 시 `guestEventEmblemBase64` 페이로드를 서버로 전송.
- 메뉴 모드 드롭다운(일반/행사) 선택 시 엠블럼 설정 영역의 노출 상태 연동.

### 2. 설정 시트 저장 (`gas/60_Settings.gs`)
- `updateGuestSettings` 및 `buildGuestSettingsResponse` 함수에 `guestEventEmblemBase64` 키 추가.
- Base64 문자열이 시트에 다이렉트로 업데이트/로드되므로 구글 드라이브 I/O 지연이나 파일 권한 문제가 완벽히 차단됨.
- 기본 설정 딕셔너리에 `guestEventEmblemBase64: ''` 초깃값 주입.

### 3. 손님 앱 동적 렌더링 (`guest.html`, `js/guest.js`)
- `guest.html` 헤더의 로고 이미지 태그에 `id="guest-app-brand-img"` 부여.
- `updateGuestAppTitle(settingsRes)`에서:
  - 행사 모드(`settingsRes.guestMenuMode === 'event'`)이고 서버에서 전달된 `guestEventEmblemBase64` 값이 존재할 경우 해당 Base64 데이터를 `img.src`에 직접 바인딩.
  - 엠블럼 값이 없거나 일반 배달 모드로 복귀 시, 기본 `icons/guest-192.png`로 즉시 원복 처리.
