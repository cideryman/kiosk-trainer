# P41 관리자 화면 게스트/행사 간식 및 순서 분리 워크스루

관리자 페이지(`admin.html`)에서 기존에 하나의 그룹(`게스트/행사`)으로 묶여 인지 부조화를 일으키던 간식 목록 및 순서 관리 영역을 **`일반 게스트`**와 **`행사/캠페인`** 2개의 독립 섹션으로 깔끔하게 분리했습니다.

---

## 🛠️ 주요 변경 내역

### 1. 간식 및 재고 관리 탭 (Tab 3: `admin.html`, `js/admin.js`)
- 기존 2개 그룹(`판매중 일반/키오스크`, `판매중 게스트/행사`)을 **3개 독립 그룹**으로 확장 렌더링하도록 개편했습니다:
  - 🛒 `판매중 키오스크 회원 간식` (`user`)
  - 🛵 `판매중 일반 게스트 간식` (`guest`)
  - 🎉 `판매중 행사/캠페인 간식` (`event`)
- 숨김 간식 보기(`toggle-hidden-snacks`) 설정 시에도 3개 독립 그룹(`숨김 키오스크 회원`, `숨김 일반 게스트`, `숨김 행사/캠페인`)으로 분리하여 표시됩니다.

### 2. 간식 표시 순서 관리 탭 (Tab 4: `admin.html`, `js/admin.js`)
- 기존 2개 순서 조정 패널에 **`🎉 행사/캠페인 간식 순서`** 독립 패널 (`id="event-snack-order-list"`)을 신설했습니다.
- `renderSnackOrderPanel`, `resetSnackOrder`, `saveSnackOrderToServer`를 3개 타겟(`user`, `guest`, `event`)에 맞게 연동했습니다.

---
