# 에셋 출처와 사용 원칙

[핸드오프로 돌아가기](../../handoff.md)

## 배달왔삼 캐릭터와 이미지

- ChatGPT와 Google Gemini의 생성형 이미지 도구를 활용해 제작했습니다.
- 신청 페이지의 `assets/delivery-banner.png`, `assets/meaning-1.jpg`, `assets/meaning-2.jpg`, `assets/meaning-3.jpg`는 OpenAI ChatGPT 이미지 생성 도구로 제작했습니다.
- `assets/dalgomi_thumb.png`, `assets/dalgomi_delivery.png`, `assets/dalgomi_heart.png`, `assets/dalgomi_cheer.png`는 구글 제미나이(Google Gemini)로 제작한 360px 크기의 공식 마스코트 '달곰이' 감정 이모티콘 4종입니다. 사용자가 직접 원하는 화풍으로 재생성하여 적용했습니다.
- 영주시 장애인복지관 주간보호 배달왔삼 프로젝트에서 기획, 선택, 수정했습니다.
- 외부 유명 캐릭터, 상표, 라이선스 이미지를 원본으로 복제하지 않습니다.

## 로고와 아이콘

- 배달왔삼 및 삼각지 매점 로고는 자체 기획·제작했습니다.
- UI는 별도 외부 아이콘 패키지 없이 프로젝트 이미지와 Unicode 이모지를 주로 사용합니다.

## 주문 알림 음원

- `sounds/`의 주문 알림은 기관 담당자와 훈련생이 자체 녹음·편집했습니다.
- 상업 음원을 사용하지 않습니다.

## 폰트

- 별도 웹폰트 CDN이나 폰트 파일을 배포하지 않습니다.
- 기기에 Pretendard가 설치되어 있으면 사용하고, 없으면 `Apple SD Gothic Neo`, `Malgun Gothic`, `맑은 고딕`, 시스템 기본 sans-serif 폰트를 사용합니다.

## 이미지 운영 규격

- 관리자에서 새로 등록하는 JPEG, PNG, WebP 이미지는 업로드 전에 최대 800px, WebP 품질 0.82로 변환합니다.
- GIF는 애니메이션 손실을 막기 위해 원본 형식을 유지합니다.
- 기존 Drive 이미지를 교체할 때 같은 파일을 새 버전으로 교체하면 연결 ID를 유지할 수 있지만, 새 파일로 다시 업로드하면 Drive ID와 주소가 바뀔 수 있습니다.

## 운영 설명서 캡처

- `docs/manual-assets/`의 관리자·주방 모바일 캡처는 현재 화면 사용법을 설명하기 위한 문서용 이미지입니다.
- 캡처는 실제 운영 화면에서 가져온 것으로, 화면 구조가 변경되면 `docs/admin-manual.md`와 함께 갱신합니다.
- 캡처에는 공개 주소에서 사용하는 닉네임 예시만 포함하며, 관리자 키·연락처·신청 개인정보는 포함하지 않습니다.
